# Hi there 👋 SoDam AI Studio 입니다

> AI와 함께(바이브코딩) 만드는 1인 스튜디오입니다. 🤖

- 🧰 **비개발자용 실용 웹/도구** — 정부 지원금 찾기, 1인용 칸반 보드 등
