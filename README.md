# 👋 SoDam AI Studio

> **AI와 함께 소프트웨어를 만드는 1인 개발 스튜디오**

- 🚀 아이디어를 빠르게 **실제 작동하는 프로그램**으로 만듭니다
- 🧩 웹 · 데스크톱 앱 · 자동화 도구까지 **분야를 가리지 않고** 개발합니다
- 📚 필요한 기술은 **바로 배워서 적용**하는 걸 좋아합니다
- 🤖 AI를 적극 활용해 **혼자서도 빠르게** 결과물을 만들어냅니다
